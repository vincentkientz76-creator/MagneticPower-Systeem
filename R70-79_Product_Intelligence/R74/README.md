# R74 — Magnetic Powerbanks Specialist

Deze regel bepaalt of een product geschikt is als **magnetische powerbank**.

---

# Criteria

- Magnetisch (R71)  
- Functie = powerbank (R70B)  
- Capaciteit ≥ 3000 mAh  
- Geen verboden kenmerken (zoals ingebouwde kabels van slechte kwaliteit)  

---

# Output

- GO  
- REVIEW  
- EXCLUDE
