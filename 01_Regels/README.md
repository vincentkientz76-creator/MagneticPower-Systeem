# 01_Regels — Canonieke MagneticPower Regels (R70–R195)

Deze map bevat alle formele regels die het MagneticPower Systeem aansturen.  
Elke regel heeft één verantwoordelijkheid en mag nooit buiten zijn domein treden.

## Waarom regels?

- Ze voorkomen chaos.
- Ze maken het systeem schaalbaar.
- Ze zorgen dat AI‑agents nooit improviseren.
- Ze beschermen de DGA‑strategie.
- Ze maken beslissingen herleidbaar.
