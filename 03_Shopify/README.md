# 03_Shopify — Theme, Templates & Implementatie

Deze map bevat alle Shopify‑specifieke implementaties van het MagneticPower Systeem.

Shopify is **uitvoerend**, nooit leidend.  
Alle beslissingen komen upstream uit de regels.

## Inhoud van deze map

### 1. theme/
Bevat theme‑bestanden, secties en snippets.

### 2. templates/
Bevat:
- collectie‑templates,
- hub‑templates,
- PDP‑templates.

### 3. metafields/
Bevat de volledige metafield‑registry (R160).

### 4. collections/
Bevat collectie‑structuren en mapping.

### 5. pdp/
Bevat PDP‑structuur volgens R174.

### 6. hubs/
Bevat hub‑kaarten en UX‑blokken.

## Relatie met regels

- R160: metafields
- R170: System Director
- R171: UX/CRO
- R172: SEO‑intentie
- R173: theme‑governance
- R174: PDP‑structuur
- R180: Shopify Build Copilot

## Doel van deze map

- Shopify consistent houden
- Theme‑wijzigingen beheersbaar maken
- UX en SEO schaalbaar maken
